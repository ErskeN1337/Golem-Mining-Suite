Golem Mining Suite

Golem Mining Suite is an easy-to-use tool for Star Citizen players who want to optimize their mining activities. It helps you locate minerals in-game and provides information about their value at different station terminals, making it easier to plan profitable mining routes.

This project is still in development, and more features will be added over time as I continue learning and improving my coding skills.
